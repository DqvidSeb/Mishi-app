# Read us letters

Create one file per letter in this folder using the `.letter.txt` extension.

Example:

```text
author: David

Hola amor,

Este es el contenido completo de la carta.
Puedes escribir varias líneas sin escapar saltos.
```

Notes:

- The file name controls the order alphabetically in the grid.
- The `author:` line is shown above the mini card.
- Everything after that line becomes the body of the letter.
